# Installation
Install node.js

    http://nodejs.org/

Clone repo

    git clone git@github.com:svagi/Game-of-credit.git

or

    https://github.com/svagi/Game-of-credit.git

Install CoffeeScript

    npm install coffee-script -g

Then install all dependencies

    npm install

Finally run application with

    npm start

or

    coffee bin/www
